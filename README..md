#  Fakebook Post Generator

## 📌 Description

This project simulates a simplified “Fakebook” experience where users can open a profile modal that displays their information, create posts with text and images, and see their posts in a feed.

This project was created as an assignment focused on classes, DOM manipulation, modular JavaScript, event handling, and dynamic HTML generation. 


## ✨ Features

- **Profile Modal**
  - Opens when clicking the profile photo  
  - Displays user information dynamically  
  - Closes automatically when the mouse leaves the modal container  

- **Post Creation**
  - Write text posts  
  - Upload an image and display the uploaded file name . 
  - Reset all fields after posting.  

- **Dynamic Feed**
  - New posts appear at the top  
  - Each post includes:
    - Full name  
    - Auto‑formatted date  
    - Text content or/and image  

- **Clean UI Behavior**
  - Modal opens and closes using click and mouseleave event listeners, respectively.
  - File name clears after posting  
  - Post validation: requires text input or image upload.
  

---

## 🛠️ Technologies Used

- **HTML** — structure and semantic layout  
- **CSS** — styling and layout  
- **JavaScript** — behaviour, DOM manipulation, modules
